cardio vigilant box

project description

cardio vigilant box is used to monitor heart rate and blood pressure continuously. It is a wearable device that can be worn on the wrist like a watch. The device uses sensors

Key Features

  1. Real-time heart rate tracking for accurate fitness monitoring
  2. Sleep tracking to understand rest patterns
  3. Notification alerts for calls, messages, and apps
  4. Long-lasting battery life, suitable for all-day usage
  5. Water-resistant design for safe outdoor and workout use
